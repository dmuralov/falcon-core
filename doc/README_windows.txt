Falcon Core

Intro
-----
Falcon is an open source, decentralized privacy platform
built for global person to person eCommerce.

For more information, as well as an immediately useable, binary version of
the Falcon Core software, see https://falcon.io.


Setup
-----
Unpack the files into a directory and run falcon-qt.exe.

Falcon Core is the original Falcon client and it builds the backbone of the network.
However, it downloads and stores the entire history of Falcon transactions;
depending on the speed of your computer and network connection, the synchronization
process can take anywhere from a few hours to a day or more.

See the falcon wiki at:
  https://falcon.wiki/
for more help and information.
